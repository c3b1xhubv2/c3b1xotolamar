Panduan singkat

Cukup 3 langkah: buat App Password Gmail, simpan di Pengaturan, lalu kirim lamaran. Setelah App Password tersimpan sekali, kamu tinggal kirim-kirim saja.

1 · Buat App Password Gmail

App Password adalah kode 16 huruf khusus dari Google. Bukan password login Gmail-mu.

1
Aktifkan Verifikasi 2 Langkah di akun Google: myaccount.google.com/security
2
Buka myaccount.google.com/apppasswords
3
Beri nama bebas (mis. Auto Lamaran), lalu klik Create / Buat.
4
Google menampilkan 16 huruf, mis. abcd efgh ijkl mnop. Salin tanpa spasi menjadi abcdefghijklmnop.
Kalau menu App Password tidak muncul, berarti Verifikasi 2 Langkah belum aktif — aktifkan dulu di langkah 1.

2 · Atur di tab Pengaturan

1
Buka tab Pengaturan.
2
Isi Email Gmail pengirim (mis. opcebi@gmail.com) — ini akun yang mengirim email.
3
Tempel App Password 16 karakter tadi di kolom App Password Gmail.
4
Isi Nama tampilan pengirim — ini nama yang dilihat HRD sebagai pengirim.
5
Klik Simpan konfigurasi.
6
Klik Tes koneksi — harus muncul hijau "Koneksi Gmail berhasil".
App Password disimpan terenkripsi di akunmu. Saat mengubah data lain nanti, kolom App Password boleh dikosongkan — password lama tetap dipakai.

3 · Kirim lamaran

1
Buka tab Kirim Lamaran.
2
Isi Email tujuan (email HRD/perusahaan) dan Subject.
3
Isi Posisi, lalu pilih Deskripsi keahlian (Bartender / Engineer / tulis sendiri / kosong). Isi email tersusun otomatis.
4
(Opsional) buka Data pengirim untuk mengubah nama/HP/email di tanda tangan.
5
Tambahkan lampiran: upload file (bisa banyak) dan/atau Tambah link file.
6
Cek Preview Email (masih bisa diedit manual), lalu klik Kirim email.

Tentang lampiran

A
Upload file langsung: total maks ~4 MB (batas Vercel). CV PDF biasa aman.
B
File besar / banyak: pakai Tambah link file. Server mengunduh dari link lalu melampirkannya sebagai file asli — tidak kena batas 4 MB.
C
Link Google Drive: set akses ke "Siapa saja yang memiliki link". Otomatis diubah jadi unduhan.

Kalau ada masalah

"ENCRYPTION_KEY harus 64 karakter"
Env server belum diatur. Admin isi ENCRYPTION_KEY lalu redeploy.
Tes koneksi "Invalid login"
App Password salah atau memakai password Gmail biasa. Buat ulang App Password.
Daftar tapi tidak bisa masuk
Konfirmasi email masih aktif — cek email untuk link konfirmasi.
"Total file terlalu besar"
Total upload melebihi ~4 MB. Gunakan opsi link untuk file besar.
"Sesi tidak valid"
Keluar lalu login ulang.
